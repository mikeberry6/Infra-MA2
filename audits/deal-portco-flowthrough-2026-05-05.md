## Review Rules

- Platform-first scope: standalone operating companies/platforms can be queued as new PortCo candidates; asset bundles, projects, portfolios, concessions, individual facilities, and minority financial stakes are usually deal-only.
- Bolt-ons through an existing platform should usually become milestones and source links on that existing platform scorecard.
- Deals through a named platform that is missing from the PortCo database are queued as platform review items first; the acquired asset is not added by default.
- Sales and IPOs only become realization/status changes when the seller's ownership period is confirmed; otherwise they are review items.
- Any new PortCo candidate requires user approval before data is added.
